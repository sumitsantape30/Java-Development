These things will be necessary to design and API.

This template is a structured way to document the key details of an API endpoint, making it easier for developers to 
understand how to interact with the API by providing information about its URL, required parameters, request body, and the type of 
operation it supports.

Every request-making URL has: 
1. Base- URL : www.google.com (Domain name)
2. API Path: -------------/complete/search
3. Request Parameter

Type Of Request(Operations): GET, POST, PUT, DELETE, PATCH
